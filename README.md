# Tic-Tac-Toe

A game of XO made with python

## Installation

Use git to clone the repo.

```bash
git clone https://github.com/Enderboy2/Tic-Tac-Toe.git
```

## Usage
Get into the directory
```bash
cd ./Tic-Tac-Toe/
```
Run the program
```bash
python main.py
```
